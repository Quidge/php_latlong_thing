This form will take in a latitude and longitude input, run it through a PHP script, and return a status page showing how far it is from the WeDoWorldWide office location (35.316770, -82.461131).

Note that there isn't any validation on this form field. If you were to input malicious code (or anything, really) to the field above it could execute. It will also break if you do anything other than two comma separated integers. It's completely possible to do on-page script execution and validation too, but I didn't get that far. Those things are covered here.
